# 📊 Unlocking Customer Insights: A Statistical Investigation

## 📌 Project Overview
This project focuses on analyzing customer data using statistical techniques to uncover behavioral patterns, spending trends, and demographic insights. The goal is to support data-driven business decisions for a retail company.

---

 ## 🎯 Objectives
- Perform Exploratory Data Analysis (EDA)
- Analyze customer demographics and behavior
- Visualize key patterns and trends
- Conduct hypothesis testing
- Generate actionable business insights

---

## 📂 Dataset Description
The dataset contains customer-level information including:

- **CustomerID** – Unique identifier  
- **State** – Customer location  
- **Education** – Education level  
- **Gender** – Gender category  
- **Age** – Age of customer  
- **Married** – Marital status  
- **NumPets** – Number of pets  
- **MonthlySpend** – Monthly spending (USD)  
- **DaysSinceLastInteraction** – Engagement metric  

---

## 🛠️ Tools & Libraries
- Python  
- Pandas  
- NumPy  
- Matplotlib & Seaborn  
- SciPy  

---

## 📊 Analysis Performed

### 1. Descriptive Statistics
- Mean, median, standard deviation
- Mode for categorical variables

### 2. Data Visualization
- Histograms & Boxplots  
- Bar charts for categorical variables  
- Scatter plots & KDE plots  

### 3. Bivariate Analysis
- Correlation matrix  
- Crosstab analysis  
- Grouped statistics  

### 4. Hypothesis Testing
- **T-test** → Gender vs Monthly Spend  
- **ANOVA** → Education & State vs Spend  
- **Chi-square test** → Marital Status vs Pets  
- **Correlation** → Age vs Engagement  

---

## 📈 Key Insights
- Education level influences spending behavior  
- Spending varies significantly across states  
- No significant difference in spending by gender  
- Age does not impact customer engagement  
- Marital status is related to pet ownership  

---

## 💡 Business Recommendations
- Target high-spending segments based on education  
- Use region-specific marketing strategies  
- Focus on behavioral data instead of age-based targeting  
- Personalize campaigns using lifestyle indicators
